# Финализированный блок по высоте

<div data-fastnear-content="endpoint-description">

Получите документ финализированного блока для заданной высоты. В зависимости от топологии деплоя запрос может быть перенаправлен на хост, которому принадлежит канонический архивный диапазон. Этот встраиваемый пример начинается с высоты генезис-блока для выбранной сети, поэтому запрос можно запустить сразу.
