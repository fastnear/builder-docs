# Агенты на FastNear

Поверхности FastNear хорошо подходят для ИИ-агентов, сред автоматизации и серверных инструментов, но точка входа зависит от задачи. Три руководства ниже помогут направить агента к нужной поверхности и сохранить его учётные данные в безопасности.

## Выбор поверхности

Сначала сопоставьте вопрос агента (состояние аккаунта, история, балансы, почти реальное время) с одной поверхностью FastNear, а потом уже открывайте ему эндпоинты. Таблица соответствий — на странице [Выбор подходящей поверхности](/agents/choosing-surfaces).

## Учётные данные для агентов

Агенты должны использовать тот же подход, что и продакшен-бэкенды: ключи в переменных окружения или менеджере секретов, а не в браузерном `localStorage`. Полный чек-лист — на странице [Аутентификация для агентов](/agents/auth).

## Шаблоны сценариев

Три частых сценария — исследование аккаунта, расследование транзакций и мониторинг почти в реальном времени — раскладываются в конкретные комбинации поверхностей. См. [Сценарии для агентов](/agents/playbooks).
