# Плейбуки для агентов

## Исследование аккаунта

- Начните с [FastNear API](/api), если вам нужны балансы, NFT, стейкинг и разрешение по публичному ключу.
- Переходите на [Справочник RPC](/rpc) только тогда, когда нужен канонический уровень деталей состояния, которого нет в индексированном представлении.

## Исследование транзакции

- Начните с [Транзакции API](/tx) для истории аккаунта, поиска по блокам и расследования по квитанциям.
- Возвращайтесь к [Справочнику RPC](/rpc) для канонической проверки статуса или сценариев, связанных с отправкой транзакций.

## Мониторинг почти в реальном времени

- Опросите [NEAR Data API](/neardata), если нужны оптимистичные или недавно финализированные чтения блоков.
- Логику опроса держите явной. Это не поверхность WebSocket или webhook.

## Поднятие инфраструктуры оператором

- Используйте [Снапшоты](/snapshots/), когда сценарий связан с поднятием инфраструктуры, а не когда агенту просто нужны данные из цепочки.
