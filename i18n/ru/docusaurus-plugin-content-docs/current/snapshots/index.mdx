# Снапшоты блокчейна

:::warning[Бесплатные снапшоты устарели]
Бесплатные снапшоты данных nearcore больше не выпускаются.

Infrastructure Committee и Near One рекомендуют использовать Epoch Sync и децентрализованную синхронизацию состояния. Подробности есть на <Link to="https://near-nodes.io">NEAR Nodes</Link>.
:::

Скачайте состояние блокчейна, если хотите быстрее поднять валидаторский или RPC-узел. Подробнее о требованиях к узлам и запуску смотрите в <Link to="https://github.com/near/nearcore?tab=readme-ov-file#about-near">nearcore</Link>.

:::info[Установка rclone]
Для этих крупных загрузок используется `rclone`, который можно установить так:

```bash
sudo -v ; curl https://rclone.org/install.sh | sudo bash
```
:::

Более подробные рекомендации по настройке валидаторов доступны на <Link to="https://near-nodes.io">NEAR Nodes</Link>.

Эти руководства используют ту же логику, что и <Link to="https://github.com/fastnear/static">репозиторий FastNear static</Link>.
